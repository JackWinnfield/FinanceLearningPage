# Révisions Finance

Base de connaissances en finance, publiée automatiquement avec [MkDocs Material](https://squidfunk.github.io/mkdocs-material/) sur GitHub Pages.

## Ajouter ou modifier une fiche

1. Crée ou édite un fichier `.md` dans le dossier `docs/`.
2. Si c'est une nouvelle page, ajoute-la au menu dans `mkdocs.yml` (section `nav`).
3. Valide (commit) : le site se reconstruit tout seul en 1–2 minutes.

## Écrire une formule

- En ligne : `\( ... \)`
- En bloc :

```
$$
S = \frac{R_p - R_f}{\sigma_p}
$$
```
